RAG Query Pipeline — 6-Node LangGraph Flow

1
Query Rewriting
Resolves pronouns, removes ambiguity, produces a clean standalone query using LLM
langchain-groq 0.2.4
llama-3.3-70b
2
PII Input Scan
Detects and redacts PII entities in the query; blocks pipeline if found
presidio-analyzer 2.2.354
spaCy 3.8.4
3
Vector Retrieval
Embeds query to 1024-dim vector; cosine similarity search in Qdrant; returns top-5 chunks
cohere 5.5.8
qdrant-client 1.9.1
4
Context Assembly
Formats retrieved chunks with source attribution; prepends Redis session history
langchain 0.3+
redis 5.0.6
5
LLM Generation
Groq API generates grounded answer from context; tokens streamed via SSE
langchain-groq 0.2.4
sse-starlette 2.1.0
6
PII Output Scan
Scrubs PII entities from LLM response before returning to client
presidio-anonymizer 2.2.354
spaCy en_core_web_sm
